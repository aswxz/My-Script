###  学习的三层界面

####  第一阶段

> 昨夜西风凋碧树。独上高楼，望尽天涯路——知道自己不懂(python是个什么玩意,都要学什么,学完能干什么,我能不能学会)

```
  到了这一境界，已经脱离了无知且不知自己无知的境界了。只有了解自己的无知，才有求知的动力，因此这是做学问的开头。
```

#### 第二阶段

> 衣带渐宽终不悔，为伊消得人憔悴——不知道自己懂(已经入门了，而不自知)

```
   已经为它费劲了心力，不断孜孜追求，把学问中重要的和不重要的都已经留在脑中，根本和枝桠都装到了脑袋里面，但是并没有意识到。
```

#### 第三阶段

> 众里寻他千百度，暮然回首，那人正在灯火阑珊处——知道自己懂了(原来就是这么个玩意)

```
这是一种顿悟，是一种去粗存精。那些看上去光鲜、美丽，但却不重要的，细枝末节的东西都被遗忘了，只留下了最本质的东西
```
